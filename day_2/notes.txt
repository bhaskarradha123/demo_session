web Technologies: the technology used to design and develop the web pages
     1.html->hypertext markup language- structure of web pages
     2.css->cascading style sheet- design/style of web pages
     3.js->javascript- logic/functionality of web pages

HTML:hypertext markup language
    :used to provide structure/layout to the web pages
    :Introduced by Tim berners-lee
    :Html files will save with .html extension

Tags: tags are used to define the structure of the web pages
    : tags are two types
             1.paired tags/container tags
             2.unpaired tags /non container tags/self closing tags

paired tags:it consist of opening tag and closing tag
            <tagName>content</tagName>
     ex: div , p, i,small,heading tags, etc

unpaired tags:it consist of opening tag only
              <tagName/>
     ex: img,br,hr, etc

element: element is a combination of opening tag , closing tag 
         and content
       <tagName>content</tagName>

comment tags: it is used to add comments in the web pages
            : the content of the comment tags will not be displayed
              in the web pages
              <!--comment content-->

attributes: it is used to add additional information to the tags
          : used in inside the opening tag
          : the content of the attributes will not be displayed
            in the web pages
        ex: src, href, alt, title,style etc

block level elements: the content of the block level elements
                     will be displayed in a new line
                     : it occupies the full width of the page
        ex: div, p, h1, h2, h3, h4, h5, h6, ul, ol, etc

inline level elements: the content of the inline level elements
                     will be displayed in the same line
                     : it does not occupy the full width of the page
                     : it occupies required space
        ex: span, a, img, input, button, etc


list: the content of the list will be displayed in a new line
